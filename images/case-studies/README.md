## Parametry

| Parametr | Hodnota |
|---|---|
| **Rozměry** | 1200 × 900 px (poměr 4:3) |
| **Formát** | JPG kvalita 80 % |
| **Velikost** | ≤ 200 KB |
| **Stylizace** | Stejný brand duotone / overlay jako blog covery (viz `images/blog/README.md`) — pro jednotnou vizuální identitu |

## Důležité: anonymizace

- Klienti potvrdili publikaci anonymizovaných údajů.
- **Žádné rozpoznatelné tváře, žádné identifikační detaily** (číslo popisné, ŠPZ auta, jmenovka u zvonku).
- Pokud focení reálných nemovitostí: vyloučit detaily, které dovolí dohledat konkrétní adresu.
- Pokud stock foto: použít neutrální bytové domy, interiéry, nebo abstraktní motivy (klíče, mapy, kalkulačka).
